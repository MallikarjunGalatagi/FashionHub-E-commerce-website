📖 How It Works

**FashionHub** is a Java-based full-stack e-commerce web application that allows users to browse fashion products, create an account, log in securely, add items to their cart, and place orders. The application is built using **Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, and JavaScript**, providing a smooth and responsive online shopping experience.
New users can register by providing their details and then log in using their **username or email** and password. Once authenticated, users can explore products by category, view product details, add products to their shopping cart, and manage their profile.
The shopping cart allows users to update item quantities or remove products before placing an order. All user, product, cart, and order information is securely stored and managed in a **MySQL database**. Session management ensures that only authenticated users can access protected features, and secure logout ends the user session safely.

🏠 Home Page

Users are welcomed with the following options:
* 🛍️ Browse Products
* 🔑 Login
* 📝 Register
* 📂 Explore Categories

📝 Registration

New users can create an account by entering:

* Username
* Email
* Mobile Number
* Password
The details are validated and securely stored in the **MySQL database**.

🔐 Login

Users can log in using:
* Username or Email
* Password

🛒 Product Catalog

Users can:
* Browse all available fashion products.
* View products category-wise.
* Check product details, pricing, and images.
* Add desired products to the shopping cart.

🛍️ Shopping Cart

The cart allows users to:
* Add products to the cart.
* Update product quantities.
* Remove unwanted items.
* View the total order amount before checkout.
  
🚪 Logout

When the user clicks the **Logout** button:
* The current session is invalidated.
* The user is redirected back to the **Home Page**.
